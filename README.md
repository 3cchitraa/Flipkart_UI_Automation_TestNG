# E-Commerce - Flipkart Automation Test Suite

This project automates test scenarios for Flipkart's product search functionality using :

---

## Features / Test Cases

- **Add To Cart Count Test**  
  Validate that the cart count updates correctly when products are added.

- **Fetch Only Few Products Test**  
  Verify that the application correctly displays a limited set of products based on filters or pagination.

- **Filter Products By Price Test**  
  Test product filtering functionality based on price ranges.

- **Flight Booking Test**  
  Validate end-to-end flight booking workflow including selection and confirmation.

- **Search Non-Existent Product Test**  
  Ensure that searching for invalid or non-existent products returns proper messages.

- **Search Product And Validate Test**  
  Search for valid products and validate their details on the product page.

- **Verify Product Title New Tab Test**  
  Open product details in a new tab and validate the product title consistency.

---

## Technology Stack

- **Language:** Java  
- **Test Framework:** TestNG  
- **Build Tool:** Maven  
- **Logging:** Log4j2  
- **Test Data:** Apache POI (for Excel)
- **Configuration Management:** Properties/config files for environment-specific settings
