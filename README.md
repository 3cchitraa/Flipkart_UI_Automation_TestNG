# Flipkart Automation Test Suite

This project automates test scenarios for Flipkart's product search functionality using :

## Prerequisites

- Java 17 or above
- Maven
- Google Chrome installed
- Internet access
- Flipkart URL
- Flipkart Account (optional)

---

## Tools & Technologies Used

- Java
- Selenium WebDriver
- TestNG
- ExtentReports
- Apache POI (for Excel)
- Maven (for build & dependencies)

---

## Setup Instructions

1. Clone the Repository
2. Import the Project
3. Install Dependencies - mvn clean install
4. Excel Test Data
